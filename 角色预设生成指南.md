# 角色预设资产生成指南

## 使用步骤

### 1. 打开Unity编辑器

确保Unity编辑器已经打开并加载了项目。

### 2. 生成角色预设资产

在Unity编辑器中，点击菜单栏：
```
Football → Generate Role Presets
```

或者按以下路径：
```
菜单栏 → Football → Generate Role Presets
```

### 3. 确认生成

点击按钮后，系统会在以下路径生成三个角色预设资产：
```
Assets/Resources/Roles/
├── ForwardRole.asset      (前锋角色预设)
├── MidfielderRole.asset   (中场角色预设)
└── DefenderRole.asset     (后卫角色预设)
```

## 如何使用生成的角色预设

### 方法1：在PlayerAI组件中配置

1. 选择场景中的PlayerAI对象
2. 在Inspector面板中找到"角色配置"部分
3. 将对应的角色预设拖拽到"Player Role"字段中：
   - 前锋 → 拖入 `ForwardRole`
   - 中场 → 拖入 `MidfielderRole`
   - 后卫 → 拖入 `DefenderRole`

### 方法2：通过代码加载

```csharp
// 加载角色预设
PlayerRole forwardRole = Resources.Load<PlayerRole>("Roles/ForwardRole");
PlayerRole midfielderRole = Resources.Load<PlayerRole>("Roles/MidfielderRole");
PlayerRole defenderRole = Resources.Load<PlayerRole>("Roles/DefenderRole");

// 分配给球员
playerAI.PlayerRole = forwardRole;
```

## 自定义角色配置

如果需要修改角色配置：

1. 在Project窗口中找到生成的角色资产文件
2. 选中文件后，在Inspector面板中查看和修改参数
3. 修改会自动保存

## 注意事项

- 角色预设资产存储在Resources文件夹下，可以通过Resources.Load()动态加载
- 每次点击生成按钮都会覆盖已有的资产文件，请谨慎操作
- 建议在修改角色配置前先备份原文件

## 故障排除

### 问题：找不到菜单"Football → Generate Role Presets"

**解决方法**：
1. 检查`Assets/Scripts/Editor/RolePresetGenerator.cs`文件是否存在
2. 在Unity编辑器中，点击`Assets → Reimport All`重新导入脚本
3. 等待Unity编译完成，菜单应该会出现

### 问题：生成后看不到资产文件

**解决方法**：
1. 在Project窗口中刷新（F5）
2. 检查`Assets/Resources/Roles/`文件夹是否存在
3. 查看Console窗口是否有错误信息

### 问题：角色配置参数不生效

**解决方法**：
1. 确认PlayerAI组件的PlayerRole字段已正确分配
2. 检查Role配置文件中的权重值是否合理
3. 在运行时检查PlayerAI的Blackboard.Role是否被正确赋值
